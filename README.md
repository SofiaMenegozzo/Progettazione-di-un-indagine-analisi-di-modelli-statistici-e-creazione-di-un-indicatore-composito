**Breve descrizione** 
Progetto per l'esame di Statistica Sociale. Unisce la progettazione di un'indagine (questionario per il rinnovo delle mense Unimib) alla costruzione di modelli statistici avanzati (Scala di Guttman su aggressività/ottimismo) e alla creazione di un indicatore composito (analisi del benessere nelle regioni italiane su istruzione, lavoro e relazioni sociali).

Questo elaborato è articolato in tre parti principali, ciascuna dedicata all’applicazione di strumenti di analisi quantitativa a contesti sociali concreti.
**Punto 1**
È stato progettato un questionario rivolto agli studenti dell’Università Milano-Bicocca con l’obiettivo di valutare la qualità delle mense universitarie in vista del rinnovo del contratto di gestione. A partire dall’albero delle aree, sono state individuate le dimensioni rilevanti del servizio (accessibilità, qualità del cibo, prezzi, organizzazione, ecc.) e sviluppate le domande della parte sostanziale del questionario, utilizzando scale di misura adeguate e facendo riferimento a fonti metodologiche sulla costruzione di survey.
**Punto 2**
È stato costruito uno scalogramma di Guttman utilizzando un sottoinsieme di item (V18_1–V18_60) per misurare uno specifico atteggiamento. Le scelte degli item sono state motivate sulla base della coerenza teorica e della cumulatività. La scala è stata validata attraverso il coefficiente di riproducibilità (CR) e l’indice di errore minimo marginale (MMR). Sono state inoltre analizzate la distribuzione della scala, le differenze di genere e la relazione con il livello di ottimismo misurato tramite il test Lot-R.
**Punto 3**
È stato costruito un indicatore sintetico a partire dagli indicatori BES (Benessere Equo e Sostenibile) per una specifica dimensione scelta tra istruzione, lavoro o relazioni sociali. Dopo aver selezionato e trattato gli indicatori elementari (inclusa la gestione dei dati mancanti), è stato definito un metodo di aggregazione con pesi motivati. L’indicatore è stato poi messo in relazione con una misura di soddisfazione soggettiva. Infine, è stata discussa una possibile strategia per aggregare più dimensioni in un unico indicatore composito di benessere.

**Struttura del Repository**
Il file elaborato.pdf contiene la trattazione teorica completa (Italiano).
Il file indicatore_sintetico.xlsx contiene i dati grezzi Istat e i fogli di calcolo relativi al Punto 3.
Il file script_R.R ed il file envoronment.Rdata contengono il codice R utilizzato per le analisi rigurdanti il Punto 2 e il Punto 3.
Il file questionario_versione_stampabile.pdf contiene il questionario creato per il Punto 1.
I file png contengono alcune immagini non direttamente aggiunte all'interno dell'elaborato, ma utili per una maggiore comprensione statistica della Cluster Analysis effettuata al Punto 3
